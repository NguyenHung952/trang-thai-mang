# 📖 Giới thiệu

**Trạng Thái Mạng** là một web tool hỗ trợ kiểm tra và theo dõi trạng thái kết nối Internet trực tiếp trên trình duyệt.

Ứng dụng cho phép:

- 🌍 Kiểm tra trạng thái kết nối mạng
- ⚡ Đo độ trễ mạng (Ping)
- 📈 Hiển thị lưu lượng mạng theo thời gian thực
- 🖥 Theo dõi phản hồi kết nối máy chủ
- 📊 Hiển thị dữ liệu trực quan bằng biểu đồ
